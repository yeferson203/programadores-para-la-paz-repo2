Sistema operativo:
Es el software principal que administra los recursos del computador o servidor. Permite ejecutar programas y gestionar el hardware.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo donde se encuentra el código del servidor. Define las rutas y las respuestas que recibirán los usuarios.

Puerto:
Es un punto de comunicación que permite al servidor recibir solicitudes. En esta actividad se utiliza el puerto 3000.

Logs:
Son registros que almacenan información sobre la actividad y eventos del sistema. Ayudan a monitorear el funcionamiento y detectar errores.
